# BTC 双币赢（Dual Investment）稳健策略实现

> ⚠️ **先说明现实限制**：任何策略都无法保证“稳定盈利”。本仓库实现的是一套**风险约束优先**的 BTC 双币赢策略框架，目标是提升长期胜率与回撤控制，而不是承诺收益。

## 策略思想（稳健版）

- 市场温和上涨/震荡：偏向 `Sell High`（持有 BTC，赚取 USDT 收益）。
- 市场温和下跌/震荡：偏向 `Buy Low`（持有 USDT，低位接 BTC）。
- 高波动、单边急涨急跌：减少或暂停开仓。
- 严格仓位上限：双币赢资金占总资产不超过配置阈值。
- 避免裸奔：只在可承受被行权的价格区间挂单。

## 功能

- 根据趋势、波动率、与资金利用率自动决定是否开仓。
- 自动在 `Buy Low` 与 `Sell High` 间切换。
- 通过参数化收益/行权模型做历史回放（简化版）。
- 输出关键指标：总收益、年化、最大回撤、胜率。

## 快速运行

```bash
python3 dual_invest_strategy.py --demo
```

这会生成一段模拟行情并执行策略，输出回测结果。

## 自定义数据回测

准备一个 CSV（按日），包含字段：

- `date`（YYYY-MM-DD）
- `close`（BTC 收盘价）

然后运行：

```bash
python3 dual_invest_strategy.py --csv your_btc_daily.csv
```

## 参数建议（偏稳健）

- `max_allocation_ratio`: `0.35`（最多 35% 资产用于双币赢）
- `vol_pause_threshold`: `0.08`（日波动率超过 8% 暂停）
- `trend_window`: `20`（趋势判定窗口）
- `base_apr`: `0.18`（基础年化估值）
- `stress_apr`: `0.10`（高波动折扣后年化）

## 重要风险提示

- 双币赢本质是“收益增强 + 条件换仓”，不是保本产品。
- 极端行情中会发生“高位卖飞”或“抄底接飞刀”。
- 真实交易请接入风控：限频、滑点、手续费、异常报价保护。
